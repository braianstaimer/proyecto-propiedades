# Diagramas de arquitectura

Cinco diagramas en **HTML + SVG puro** — sin Mermaid, sin CDN, sin dependencias externas. Cada uno es un archivo `.html` autosuficiente que abre offline. Servidos por el mismo nginx que esta docs site.

> **Cómo regenerarlos:** los archivos viven en `backend-developer/static/diagrams/` (copiados desde el workspace `assessments/diagrams/`). Editar los `<rect>`/`<line>`/`<text>` directos del SVG y commitear.
