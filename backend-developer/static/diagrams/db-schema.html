Carga inicial (PDF requirement)

Volumen montado en /docker-entrypoint-initdb.d de la imagen oficial mysql:8.0 ejecuta scripts en orden alfabético en el primer arranque (volume vacío):

./backend/persistencia/
├── 01_schema.sql          ← CREATE TABLE + grants
└── 02_seed_data.sql       ← 20 INSERTs

Para reseed: docker compose down -v && docker compose up — borra el volume y reaplica scripts.
